# GitLab Community Edition

[GitLab](https://about.gitlab.com/) is a single application for the entire software development lifecycle.

GitLab Community Edition provides source code management (Git repository), issue tracking, CI/CD, and much more.

On initial startup, GitLab may take several minutes to configure itself (especially the bundled Postgres and Redis databases). Once it is ready, access the Web UI to login and retrieve your initial root password or configure an administrative user.
